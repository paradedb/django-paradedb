# Examples

## Quick Start

```bash
# Start ParadeDB
bash scripts/run_paradedb.sh
source scripts/paradedb_env.sh

# Run an example
python examples/quickstart/example.py
```

## Available Examples

- **[quickstart](quickstart/)** - Basic full-text search with BM25
- **[rag_ollama](rag_ollama/)** - RAG with local LLM
- **[hybrid_rrf](hybrid_rrf/)** - Hybrid search (BM25 + vector) with RRF fusion
